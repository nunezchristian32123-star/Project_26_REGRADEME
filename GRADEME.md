# Project_26
Stopwatch
stopwatch starts by first declaring the lcd library so we can use it. SELECT button is defined along with none to simulate no response to the button. Variable defining the lcd button presssed are setup along with an offset to offset time added with the delay for reading the final response. if statement with (adc_key_in < 850 && adc_key_in > 651) reads the voltage of button press and outputs either SELECT or NONE depending if a button is pressed or not. Setup sets the lcd screen to allow for usage of the entire screen. Loop sets up the math for the timer starting from millis and displays the elapsed time since the program ran in ms, sec, and min. lcd_key is then defined from zero to the button pressed (select) this is used in the case to choose SELECT which stops the timer adding an offset to account for delay. NONE is selected if nothing is read from the sensor continuing the count.
